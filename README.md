# Brian Maina Portfolio

Welcome to the repository for my personal portfolio website. This project is a modern web application designed to showcase my work, resume, and services while providing a seamless user experience. 

## Features

Based on the project configuration, this portfolio includes several advanced capabilities:
* **Modern Framework:** Built with Next.js and React for fast, server-rendered pages.
* **Database Integration:** Utilizes MongoDB for robust data storage and retrieval.
* **Smooth Animations:** Integrated with Framer Motion to provide engaging user interface transitions.
* **Custom Styling:** Styled using Tailwind CSS alongside UI components from Shadcn UI.
* **Communication:** Email sending and templating handled via Resend and React Email.
* **PDF Handling:** Built-in support for generating and previewing PDFs using react-pdf and jspdf.
* **Theme Support:** Dark and light mode toggle enabled via next-themes.

## Tech Stack

* **Frontend:** Next.js (v16), React (v19), Tailwind CSS (v4)
* **Backend / Database:** Next.js API Routes, MongoDB
* **UI / Animations:** Framer Motion, Lucide React, Shadcn UI
* **Utilities:** Resend (Emails), Upstash (Rate Limiting), Bcryptjs (Security)

## Getting Started

First, ensure you have the required environment variables set up for your database and email services. Then, install the dependencies and run the development server:

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. 

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
